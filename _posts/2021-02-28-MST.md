# MST 알고리즘
> 대표적으로 Prim 과 Kruskal 알고리즘이 존재한다.
> sparse한 형태의 그래프는 Kruskal 알고리즘이 적합하고 밀도 높은 Dense한 그래프에는 Prim 알고리즘이 적합하다.   

## 스패닝 트리
> 그래프 내의 모든 정점을 지나가는 트리를 의미한다. 보통 최소 연결 부분 그래프를 의미한다.

## Prim : O(N^2)
<code>
    def solve(start):
        heap=[]
        visit=[0]*(v+1)
        visit[start]=1
        cnt=1
        d=0
        for w,n in graph[start]:
            heappush(heap,(w,n))
        
        while heap:
            w,cur = heappop(heap)
            if not visit[cur]:
                visit[cur] = 1
                cnt+=1
                d+=w
                for weight,nxt in graph[cur]:
                    heappush(heap,(weight,nxt))
            if cnt == v:
                return d
        return 0
</code>

## Kruskal : O(eloge)
<code>
    def find(node):
        if parent[node] == node:
            return node
        else:
            parent[node]=find(parent[node])
            return parent[node]

    def union(a,b):
        a = find(a)
        b = find(b)
        if a != b:
            parent[b]=a

    def kruskal():
        mst =0
        cnt =0

        for e in graph:
            a,b,w = e
            if find(a) != find(b):
                union(a,b)
                mst+=w
            if cnt == v-1:
                break
        return mst
</code>
